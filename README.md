# Gulp
=====

## Get Started

Require Node.js v8.11.3

```
ndenv install 8.11.3
ndenv rehash
```

## Usage
### development
```
npm start
```

### build
```
npm run build
```

### release
```
npm run release
```

## Gulp Task

### CSS Build
```
gulp css:preprocessor
gulp css:minify
```

### JS Build
```
gulp js
gulp js:minify
```

### Image Sprite
```
gulp image:sprite
```

### Image Minify
```
gulp image:minify
```
